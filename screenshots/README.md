# Screenshots

Letakkan file screenshot di sini dengan nama-nama berikut:

1. **preview.png** - Screenshot utama (tampilan awal dengan kamera aktif)
2. **home.png** - Halaman utama
3. **gesture.png** - Saat mendeteksi gesture
4. **album.png** - Tampilan album foto
5. **theme.png** - Modal customisasi warna

## Cara Ambil Screenshot
1. Buka website di browser
2. Tekan `F12` untuk buka DevTools
3. Toggle device toolbar (Ctrl+Shift+M) untuk mode mobile
4. Screenshot dengan `Ctrl+Shift+S` atau klik kanan → Screenshot

## Format yang Disarankan
- Ukuran: 1280x720 atau 750x1334 (mobile)
- Format: PNG atau JPG
- Kualitas: Tinggi
